后台管理二次开发说明

一、后台菜单规则
左侧显示方式为：
一级菜单
├─┬ 二级菜单
│ ├─ 三级菜单
│ └─ 三级菜单
└─┬ 二级菜单
  └─ 三级菜单

1.一级菜单（顶部菜单）
2.二级菜单（左侧分类菜单）
3.三级菜单（左侧可点击菜单）
4.四级菜单（不做显示用，用于权限控制）

二、开发手册
因为本系统都是采用的开源框架，因此如果有不清楚的可以参考对应手册
1.PHP请参考ThinkPHP3.2官方手册
2.easyui请参考easyui官方手册
